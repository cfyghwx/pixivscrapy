# 使用说明
## 1.关于vpn设置说明
1）若使用shadowsock等设置本地端口代理方式的vpn，则需要自己修改middleware中的proxy
2）若使用其他的全局代理方式的vpn，注释掉middleware中的proxy

## 2.账户设置
爬取时需要设置记得的账户和密码，根据passwordsample格式设置，同时文件命名为password即可
